# Пример реализации CRUD операций с использованием веб-интерфейса и Spring Framework в Java

Этот проект представляет собой пример реализации CRUD (Create, Read, Update, Delete) операций с использованием веб-интерфейса на Java. В проекте используется компоненты Spring Framework для создания веб-приложения.

Структура проекта
src: Директория содержит исходные файлы Java.
controller: Директория содержит контроллеры для обработки HTTP-запросов.
service: Директория содержит сервисы для выполнения бизнес-логики.
dao: Директория содержит объекты доступа к данным (Data Access Objects) для взаимодействия с базой данных.
entity: Директория содержит сущности (Entity) - объекты, которые мапятся на таблицы базы данных.
view: Директория содержит файлы представлений для веб-интерфейса.
webapp: Директория содержит ресурсы веб-приложения, такие как HTML, CSS.
aspect: Директория содержит аспекты, используемые для аспектно-ориентированного программирования.
application-context.xml: Файл конфигурации контекста приложения Spring.

Использованные технологии
Spring Framework: Используется для управления компонентами приложения, внедрения зависимостей и управления транзакциями.
Spring Web MVC: Используется для создания веб-приложений с использованием шаблона проектирования MVC (Model-View-Controller).
MySQL: Реляционная база данных, используется для хранения данных приложения.
Hibernate: Используется для взаимодействия с базой данных и отображения объектов Java на записи в базе данных.
Validator: Используется для валидации данных ввода.

Требования
Для запуска этого проекта вам понадобится:
JDK (Java Development Kit) версии 8 или выше.
Среда разработки Java, такая как IntelliJ IDEA, Eclipse или любая другая.
Apache Maven для управления зависимостями.
Сервер MySQL.

Как запустить
Установите MySQL и создайте базу данных с именем employees.
Укажите настройки базы данных в файле src/main/resources/applicationContext.xml.
Клонируйте репозиторий на свой локальный компьютер:
git clone https://github.com/IlyasNasirov/spring_mvc_hibernate_aop.git
Перейдите в директорию проекта:
cd spring_mvc_hibernate_aop
Соберите проект с помощью Maven:
mvn clean package
Настройте Tomcat:
Скачайте и установите Apache Tomcat.
Скопируйте собранный WAR файл (spring_mvc_hibernate_aop.war) из директории target в директорию webapps внутри директории, где установлен Tomcat.
Запустите Tomcat, выполнив команду startup.sh (для Unix) или startup.bat (для Windows) из директории bin внутри директории Tomcat.
После запуска Tomcat, ваше веб-приложение будет доступно по адресу http://localhost:8080/spring_mvc_hibernate_aop.
Использование
После запуска приложения вы сможете взаимодействовать с ним через веб-интерфейс.
Пример
Пример использования веб-интерфейса для создания работника:
Откройте веб-браузер и перейдите по адресу http://localhost:8080/spring_mvc_hibernate_aop/allEmployees.
На главной странице нажмите на кнопку "Add".
Заполните данные нового работника и нажмите на кнопку "Submit".
Работник будет добавлен в базу данных, и вы увидите полный список всех работников.
Вы также сможете просмотреть, обновить и удалить работников через веб-интерфейс.
